# Usage

## Set up on a new machine
```
chezmoi init --apply https://github.com/<user>/dotfiles.git                                                                                                               
```
# Edit a config file
```
chezmoi edit ~/.config/nvim/init.lua                                                                                                                                    
```
# Apply chenges                                                                                                                                                           
```
chezmoi apply
```
                                                                                                                                                                            
# Show diff
```
chezmoi diff
```
